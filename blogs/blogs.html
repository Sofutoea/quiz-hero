Q1 : What is JavaScript used for?

Answer : JavaScript is used to make web pages interactive and dynamic. It can update content, validate forms, handle user events (like clicks), and create animations on websites.

Q2 : What are JavaScript data types?

Answer : JavaScript data types include string (text), number (both integers and decimals), boolean (true/false), object, undefined, and null. These help define what kind of data a variable holds.

Q3 : What is the difference between == and === in JavaScript?

Answer : The `==` operator checks if values are equal but ignores data types. The `===` operator checks both the value and the type. It's best to use `===` for accurate comparisons.

Q4 : What are JavaScript functions?

Answer : Functions are blocks of code used to perform specific tasks. They allow you to reuse the same code by calling the function multiple times.

Q5 : What is the purpose of JavaScript Promises?

Answer : Promises are used to handle tasks that take time, like fetching data from a server. They help organize code and handle success or failure of these tasks.

Q6 : What is an event in JavaScript?

Answer : Events are actions that happen on a webpage, like clicking a button, scrolling, or submitting a form. JavaScript can respond to these events by running code.

Q8 : What is the Document Object Model (DOM)?

Answer : The DOM is a representation of the structure of a webpage. It allows JavaScript to interact with and manipulate HTML and CSS elements on the page.

Q9 : What is an array in JavaScript?

Answer : An array is a collection of values stored in a single variable. These values can be accessed by their position (index) and can include different data types.

Q10 : What is a callback function?

Answer : A callback function is a function passed as an argument to another function. It is called later when the task is complete, allowing asynchronous operations.
